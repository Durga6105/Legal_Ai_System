# Legal_Ai_System
# ⚖️ AI Legal Assistant with Memory (NDA Risk Analyzer)

## 🚀 Overview

An advanced Legal AI system that analyzes NDAs and legal documents using LLMs and provides clause-level risk assessment.

## 🔥 Features

* 📄 NDA Analyzer (automatic clause checking)
* ⚠️ Risk detection (missing / risky clauses)
* 🧠 Memory system using ChromaDB
* 💬 Legal Chat Assistant
* 📊 Insights dashboard
* ⚡ Async LLM processing for speed

## 💡 Key Highlights

* Built an AI-powered legal assistant with memory
* Implemented vector database for contextual recall
* Designed async pipeline for faster document processing
* Developed clause-level risk detection system

## 🏗️ Architecture

* UI: Streamlit
* LLM Processing: Groq (LLaMA 3)
* Workflow: LangGraph
* Memory: ChromaDB (Vector Database)
